# Civilization VI

## Prerequisites

- Purchase and install Civilization VI from Steam

## Setup Instructions

1. **Launch the Game**

   Open Civilization VI from your game library.

2. **Select Tutorial Mode**

   When starting the game, select the **Beginner Tutorial Mode**. This provides a guided learning experience that is suitable for agent interaction.

3. **Run the Agent**

   Use the following command to start the agent:

   ```bash
   python scripts/play_game.py --config ./src/agent_client/configs/civilization/config.yaml
   ```

## Notes

- The tutorial mode provides a controlled environment for testing and evaluation.
- Make sure the game is running before starting the agent script.
