# Phoenix Wright: Ace Attorney Trilogy

## Prerequisites

- Phoenix Wright: Ace Attorney Trilogy installed via Steam

## Setup Instructions

1. **Install the Game**

   Purchase and install [Phoenix Wright: Ace Attorney Trilogy](https://store.steampowered.com/app/787480/Phoenix_Wright_Ace_Attorney_Trilogy/) from Steam.

2. **Enter Chapter 1**

   Start the game and progress to **Chapter 1** (第一章).

3. **Run the Agent**

   Once you are in Chapter 1, run:

   ```bash
   python scripts/play_game.py --config ./src/agent_client/configs/pwaat/config.yaml
   ```

## Notes

- No mods, plugins, or BepInEx are required. Only the Steam game is needed.
- Ensure the game is in Chapter 1 before starting the agent.
