# Baba Is You

## Prerequisites

- Baba Is You installed via Steam

## Setup Instructions

1. **Install the Game**

   Purchase and install [Baba Is You](https://store.steampowered.com/app/736260/Baba_Is_You/) from Steam.

2. **Enter Level 0**

   Start the game and enter **Level 0** (第零关).

3. **Run the Agent**

   Once you are in Level 0, run:

   ```bash
   python scripts/play_game.py --config ./src/agent_client/configs/baba_is_you/config.yaml
   ```

## Notes

- No mod scripts or extra configuration are required. Only the Steam game is needed.
- Ensure the game is in Level 0 before starting the agent.
