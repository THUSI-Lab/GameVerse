# Genshin Impact

## Prerequisites

- Purchase and install Genshin Impact from the official website https://genshin.hoyoverse.com/en/ .

## Setup Instructions

1. **Launch the Game**

   Open Genshin Impact from the official website.

2. **Create the Account**

   Create a new count and start a new journey. Start the first chapter.

3. **Run the Agent**

   Use the following command to start the agent:

   ```bash
   python scripts/play_game.py --config ./src/agent_client/configs/genshin/config.yaml
   ```

## Notes

- Make sure the game window is active before running the agent.
